# Glossary

- **Event ticker**: Kalshi identifier for an event (e.g., a specific expiry window).
- **Market ticker**: Kalshi identifier for a specific strike/market within an event.
- **Ladder**: The set of strikes \(K\) offered for a given event.
- **YES / NO**: Complementary contracts paying \(\$1\) if the event occurs (YES) or does not occur (NO).
- **Executable price**: A price you can plausibly trade at now (vs a mid or theoretical value).
- **Buy-now proxy**: Our inferred executable buy price using opposite-side bids when asks are missing.
- **Edge (pp)**: \(p_{\text{model}} - p_{\text{mkt}}\), measured in probability points.
- **EV**: Expected value per contract in dollars, net of fees, under conservative execution assumptions.
